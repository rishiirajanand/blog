# Blog-Application

## Tech Used
  - NodeJS
  - ExpressJs
  - JWT auth
  - MongoDB database
  - ReactJs

## Features
  - Login, Signup, JWT middleware, and CRUD operations using a MongoDB database (cloud).
  - Only authorized users can access the blog.
  - User can only delete/update own blog
